📌 User Dashboard (CRUD Application)
A simple user management dashboard built using HTML, CSS, and basic JavaScript functionality. This project demonstrates core CRUD operations (Create, Read, Search, Delete) with a clean and minimal UI.

🚀 Features
Add new users with name and email
Search users by name
Reset search results
Display user list dynamically
Delete users instantly
